# Sanat Merkezi Frontend

React + Vite ile geliştirilmiş modern sanat merkezi yönetim paneli.

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Uygulama http://localhost:5173 adresinde çalışacak.

## Build

```bash
npm run build
```

## Özellikler

- Modern ve premium UI/UX
- Role-based access control
- Responsive design
- Smooth animations
- Glassmorphism effects

## Sayfalar

- **Dashboard**: Genel bakış ve hızlı erişim
- **Students**: Öğrenci yönetimi
- **Teachers**: Öğretmen yönetimi
- **Courses**: Ders yönetimi
- **Schedule**: Haftalık ders programı
- **Payments**: Ödeme takibi

## Teknolojiler

- React 18
- Vite
- React Router DOM
- Axios
- Vanilla CSS (Custom Design System)
